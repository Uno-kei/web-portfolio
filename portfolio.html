First Experience
with Logic Circuit

This logic circuit was a project from the subject CPEN 75: Logic Circuit and Design. This circuit is a cube of 2 bits and it works like, if we are given an input of 00 which is equivalent to the decimal of 0, if we cube this we will also get an output of 0. If we have an input of 01 which is equivalent to the decimal of 1, the cube of this is equal to 1 with an output of 00001. For input 10 equivalent to 2 decimal, the cube of this input is equal to 8 with an output of 01000. Lastly if we have both inputs of 1 it will have an equivalent of 3 with an output of 11011 or 27 in decimal.
